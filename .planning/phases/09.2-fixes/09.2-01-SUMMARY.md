# Phase 09.2 Plan 01: Refined Header and Model Control Bar Summary

## Overview

Refactored the dashboard layout to separate model/chat controls into a dedicated second bar, leaving the top bar for window/system controls and drag operations. This provides a clearer separation of concerns and a more spacious layout.

## Key Changes

### UI / UX
- **Two-Bar Header Structure**:
    - **Top Bar**: Now focused on system controls (Sidebar toggle, Incognito toggle, Theme switch, Window controls).
    - **Second Bar**: Contains the horizontally centered `ModelSelector`, an "Unload" button for local models, and the Right Sidebar toggle.
- **Unload Functionality**: Added a `LogOut` (used as Eject) icon that appears when a local model is selected, allowing users to clear the current selection.
- **Responsive Positioning**: The second bar is sticky and positioned below the main header.
