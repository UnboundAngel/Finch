# Phase 09.4 Plan 01: Header Consolidation Summary

Consolidated the dashboard header from two separate bars into a single, compact h-14 strip. This merge places chat-specific controls (Model Selector, Eject button) in the center, while sidebar toggles and system controls (Incognito, Theme, Window Controls) occupy the left and right wings.

## Accomplishments

- **Unified Header Strip**: Removed the `SECOND BAR` and integrated its functionality into the `TOP BAR`.
- **Improved Centering**: Used `flex-1` columns to ensure the Model Selector remains perfectly centered regardless of the number of items on either side.
- **Local Model Eject**: Re-implemented the Eject button using the specific `/assets/eject.svg` asset, appearing conditionally for local model providers.
- **Optimized Drag Regions**: Moved `no-drag` classes from container divs to specific interactive elements (buttons, selectors), ensuring that the background areas of the header remain draggable for Tauri window movement.
